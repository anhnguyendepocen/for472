Tree height and diameter data from the Aspen FACE Experiment, 1997-2008

Research Data Product File Index

File Folder Description
_metadata_RDS-2013-0015.html   Metadata file in HTML format containing a description of the content, quality, and other characteristics of the data.
_metadata_RDS-2013-0015.xml   Metadata file in Extensible Markup Language (XML) format containing a description of the content, quality, and other characteristics of the data.
FACE_aspen_core_growth.csv \Data Comma-delimited ASCII text file containing tree height and diameter measurements for the east half of each ring, which was aspen monoculture.
FACE_aspen-birch_core_growth.csv \Data Comma-delimited ASCII text file containing tree height and diameter measurements for the northwest corner of each ring, which was planted with a 50-50 mixture of aspen clone 216 and paper birch.
FACE_aspen-maple_core_growth.csv \Data Comma-delimited ASCII text file containing tree height and diameter measurements for the southwest corner of each ring, which was planted with a 50-50 mixture of aspen 216 and sugar maple.
Additional_Info.pdf \Supplements Adobe Acrobat PDF/a file containing additional information on methods and quality assurance / control checks recorded by date across the measurement years.
RingMaps.pdf \Supplements Adobe Acrobat PDF file containing the 12 rings maps (and legend) for the Aspen FACE experiment. The maps are labelled as 'replicate.treatment' where replicate=1, 2, or 3 and treatment = 1, 2, 3, or 4. (Replicate 1=north, 2=central, and 3=south. Treatment 1=control, 2=elevated CO2, 3=elevated O3, and 4=elevated CO2+O3)
